# YouTube-Trending-Analysis-Excel

Project Overview

This project analyzes YouTube Trending Videos in the United States using Microsoft Excel. The objective was to identify popular content categories, channel performance, audience engagement patterns, and characteristics of videos appearing on the trending page.

The dashboard was built using Pivot Tables, Pivot Charts, Slicers, and Excel Dashboarding techniques.

Pivot Tables

Business Questions Answered
Which categories dominate YouTube Trending?
Which channels appear most frequently on the trending page?
Which categories generate the highest views?
Which categories receive the highest audience engagement?
How do likes and dislikes vary across categories?
Which videos generated the highest views?
Which categories have the best Like-to-Dislike ratio?
Dashboard Components
Trending Videos by Category

Shows the percentage distribution of trending videos across different content categories.

Top Categories by Total Views

Identifies categories generating the largest share of total views.

Top Channels Dominating Trending

Highlights channels with the highest number of appearances on the trending page.

Like vs Dislike Analysis

Compares audience sentiment across major content categories.

Comment Engagement Analysis

Measures audience interaction through comment activity.

Average Like-to-Dislike Ratio

Evaluates audience approval rates by category.

Top Viewed Videos

Displays the highest-performing videos in the dataset.

Key Insights

Based on dashboard analysis:

Music dominates YouTube Trending, contributing more than 50% of total views.
Entertainment ranks second in overall audience reach.
Music videos receive the highest engagement levels.
Certain channels repeatedly appear on the trending page, indicating strong audience retention.
Music and Entertainment categories show the strongest audience approval ratios.
A small number of viral videos account for a significant portion of total views.
